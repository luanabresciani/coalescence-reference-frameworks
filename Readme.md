# Coalescence Reference Frameworks

This repository contains the code used to implement the Balanced Averaging Null Model (BANM) and the Balanced Donor-Sum (BDS) framework developed to evaluate microbial community coalescence outcomes. These reference frameworks generate expected community compositions under alternative assumptions regarding donor contributions and community reassembly.

The BANM framework uses permutation-based randomization of donor replicates while enforcing balanced donor contributions, generating null expectations in the absence of deterministic donor-driven reassembly. In contrast, the BDS framework directly combines donor communities while preserving donor-derived compositional structure, providing a reference expectation based on simple donor blending.

Both frameworks produce simulated coalescence communities that can be used in downstream ecological analyses, including community composition, diversity, stability, and network analyses.
